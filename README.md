## ⚡ How to Use

1. **Clone the repository:**
    ```bash
    git clone https://github.com/InesZenkri/OpenSAP-Python-Course.git
    cd OpenSAP-Python-for-Beginners
    ```

2. **Navigate to the desired solution:**
    ```bash
    cd solutions/Week_1
    ```

3. **Execute the Python script:**
    ```bash
    python exercise_1.py
    ```
